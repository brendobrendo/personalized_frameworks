# Bank of Bargo Interaction Framework (Checklist)

---

## 🟦 1. Pre-Interaction (30–60 sec)
- [ ] Open **Salesforce CRV**
- [ ] Review Offers + GPCs
- [ ] Scan account types / balances
- [ ] Check recent notes / interactions
- [ ] Identify 1–2 potential conversation angles

---

## 🟦 2. Open + Authenticate
- [ ] Greet naturally
- [ ] Authenticate via **CRV (not NEP)**
- [ ] Ask primary question: “What can I help you with today?”
- [ ] Ask expansion question: “Anything else you wanted to look at?”

---

## 🟦 3. Handle Primary Need
- [ ] Complete requested task
- [ ] Listen for signals:
  - [ ] Large balances → Investment convo
  - [ ] Credit activity → Card convo
  - [ ] Life events → Referral opportunity

---

## 🟦 4. Expand (Offers + GPCs)
- [ ] Transition to advisory:
  - “While I have your profile up…”
- [ ] Present **1–2 relevant products max**
- [ ] Tie recommendations to customer situation

### Required CRV Actions
- [ ] Disposition ALL Offers
- [ ] Disposition ALL GPCs

---

## 🟦 5. Create Referral (FA Focus)
- [ ] Check for triggers:
  - [ ] $25k+ idle cash
  - [ ] Retirement questions
  - [ ] Large deposit / liquidity event
  - [ ] Expressed uncertainty about finances
- [ ] Ask for FA introduction
- [ ] Create referral in CRV (if applicable)

---

## 🟦 6. Close the Conversation
- [ ] Recap what was completed
- [ ] Reinforce value provided
- [ ] Set expectation for follow-up

---

# 🟩 7. Post-Interaction Execution

## A. Summary Note
- [ ] Reason for visit
- [ ] Actions taken
- [ ] Last 4 digits of accounts affected
- [ ] Offers/GPC outcomes
- [ ] Referrals created

---

## B. Uncovered Needs
- [ ] Document remaining opportunities
- [ ] Identify next best action

---

## C. Follow-Up Email
- [ ] Send template email
- [ ] Personalize 1–2 lines
- [ ] Reference actual conversation

---

## D. Follow-Up Opportunity
- [ ] Create opportunity in CRV
- [ ] Add **specific reason**
- [ ] Set **clear date**

---

## E. Affluent Activities
- [ ] Log if applicable

---

## F. Add to Book
- [ ] Add client (if appropriate)
- [ ] Mentally classify:
  - [ ] Builder
  - [ ] Maintainer
  - [ ] Opportunistic

---

# 🔁 Quick Mental Loop (L.E.A.D.S.)
- [ ] Lead the interaction
- [ ] Execute the task
- [ ] Advise (Offers + GPCs)
- [ ] Deepen (Referrals)
- [ ] System (CRV + follow-up)

---

# ⚡ Daily Performance Targets
- [ ] Disposition 100% of Offers + GPCs
- [ ] Create 1–2 FA referrals
- [ ] Complete all notes + follow-ups same day
